Token digunakan untuk menyimpan data feedback ke GitHub Gist. Lihat cara mendapatkan token

Status: Tidak diketahui

Gist ID: Tidak ada

Cara Mendapatkan Token GitHub:

  1. Login ke akun GitHub Anda
  2. Klik pada foto profil di pojok kanan atas dan pilih Settings
  3. Scroll ke bawah dan pilih Developer settings
  4. Pilih Personal access tokens lalu Tokens (classic)
  5. Klik Generate new token dan pilih Generate new token (classic)
  6. Berikan nama deskriptif untuk token
  7. Pilih expiration (masa berlaku) token
  8. Centang opsi gist untuk memberikan izin membuat dan mengelola gists
  9. Klik Generate token di bagian bawah halaman
  10. Salin token yang dihasilkan (ini hanya akan ditampilkan sekali)
